# 1.安裝OpenSSH: 
```
sudo apt-get install openssh-server openssh-client
``` 
or 
```
sudo apt-get install sshd
```

# 2.配置并开启ssh服务: 
```
sudo vim /etc/ssh/sshd_config
sudo service ssh restart
```

#3.查看sshd:
```
sudo netstat -tunlp |grep ssh
```
有sshd则没问题

#4.连接服务器：
```
sudo username@域名或IP -p 端口号
```
